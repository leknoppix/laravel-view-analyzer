Fonctionnalités

Comprenez l'Architecture de vos Vues

Laravel View Analyzer va au-delà du simple scan de fichiers. Il comprend comment votre application fonctionne.

Détection de Vues Inutilisées

Identifiez les templates qui ne sont plus référencés dans vos contrôleurs, routes ou autres vues.

Cartographie des Contrôleurs

Visualisez la relation entre vos contrôleurs et vos vues dans un format arborescent hiérarchique.

Tableau de Bord Web

Inclut un superbe tableau de bord intégré accessible via votre navigateur pour explorer les résultats d'analyse.

Analyse Intelligente

Détecte automatiquement les appels dynamiques, les liens de pagination, les mailables, les notifications, les commandes et les références de composants.

Démarrez en quelques secondes

Terminal

$ composer require leknoppix/laravel-view-analyzer --dev

$ php artisan vendor:publish --tag=view-analyzer-config

$ php artisan views:analyze
